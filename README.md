# 实时对战井字棋游戏

一个基于 Node.js 和 Socket.IO 构建的实时对战井字棋游戏，支持双人在线对战、实时聊天等功能。

## 功能特点

### 核心游戏功能
- ✨ 实时双人对战
- 🎮 回合制游戏机制
- 🏆 自动判定胜负
- 📊 实时比分系统
- 🔄 一键重新开始

### 玩家交互
- 💬 实时聊天系统
- 😊 表情选择器
- ⏱️ 回合计时器
- 🔔 游戏音效系统
- 👥 玩家状态显示

### 用户体验
- 🎯 清晰的游戏状态提示
- 🎨 美观的界面设计
- 📱 响应式布局
- ✨ 流畅的动画效果

## 技术栈

- **前端**：HTML5, CSS3, JavaScript
- **后端**：Node.js, Express
- **实时通信**：Socket.IO
- **开发工具**：npm
